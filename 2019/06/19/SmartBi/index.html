功能简介

  • 数据源：数据源支持大数据和关系型还有非关系

  • 数据处理：特诊离散，降维都可以

  • 可视化拖拉挖掘 ：所有算法都可以拖拉进来，都是组件
  • 支持训练优化模型的保存，可以团队复用
  • 多模型组合，多小祖协同工作
  • 支持模型流程导出导入
  • 服务发布：服务部署+ 服务监控
  • 接口管理开放还是关闭

挖掘实践流程

  1. 确定商业目标，
  2. 数据准备，建立模型，持续优化
  3. 评估模型预测效果 6成到8成
  4. 应用部署：对接口开放和关闭
